## 第一章：BER 能源等级证书全解读

### 1.1 什么是 BER

BER（Building Energy Rating）是爱尔兰法定的建筑能效评级证书，类似家电的能效标签，评级从 **A1（最优）** 到 **G（最差）**。

- **法律依据**：S.I. No. 666/2006 European Communities Regulations
- **强制场景**：出售、出租、新建房屋必须持有有效 BER
- **有效期**：10 年
- **评估机构**：SEAI（Sustainable Energy Authority of Ireland）统一管理

### 1.2 BER 评级与能耗对照

| BER 等级 | 一次能源消耗 (kWh/m²/yr) | CO₂ 排放 (kg/m²/yr) | 典型房屋年代 |
|----------|-------------------------|---------------------|-------------|
| A1-A3    | 0-75                    | 0-15                | 2020+ 新建节能房 |
| B1-B3    | 75-150                  | 15-30               | OSS 全屋升级后目标 |
| C1-C3    | 150-225                 | 30-45               | 2005-2010 建造 |
| D1-D3    | 225-350                 | 45-65               | 1990-2005 建造 |
| E1-E3    | 350-475                 | 65-80               | 1970-1990 建造 |
| F-G      | >475                    | >80                 | 老石头房 / 无保温 |

**关键门槛**：OSS 全屋升级项目要求完工后达到 **B2 或以上** + 一次能源提升至少 **100 kWh/m²/yr**。

### 1.3 BER 证书包含哪些信息

一份标准 BER Advisory Report 包含：

1. **房屋基本信息**：地址、面积、建造年份、房屋类型
2. **各部件能效评级**：
   - 墙体保温等级
   - 屋顶/阁楼保温等级
   - 地板保温等级
   - 窗户/门能效
   - 供暖系统效率（主供暖 + 控制）
   - 热水系统
   - 通风系统
   - 照明能效
   - 可再生能源（太阳能、热泵等）
3. **改善建议清单**：按"费用-效果"排序的升级建议（含预估成本、年节省、回本周期）
4. **评估师信息**：SEAI 注册号、评估日期

### 1.4 BER 评估流程

```
步骤 1：在 SEAI 官网 (seai.ie) 搜索注册 BER 评估师
步骤 2：预约上门评估（约 1-2 小时）
步骤 3：评估师现场测量并拍照记录
步骤 4：评估师提交 DEAP 软件计算结果
步骤 5：SEAI 生成 BER 证书并录入全国登记册
步骤 6：业主收到 BER 证书和 Advisory Report
```

**费用参考**：
- 普通住宅 BER 评估：€250-€400（含 VAT）
- 大型/复杂住宅：€400-€600
- OSS 全屋升级前后的两次 BER：通常包含在 OSS 报价中

### 1.5 如何查找历史 BER

爱尔兰全国 BER 登记册公开可查：
- SEAI 官网 → National BER Register → 输入 MPRN（电表号）或地址
- 如果前业主已经做过 BER，可以免费查到
- 如果没有记录，表示前业主从未请过评估师

## 第二章：OSS 全屋能源升级深度解读

### 2.1 什么是 One Stop Shop（OSS）

OSS 是 SEAI 推出的**全屋能源升级一站式服务模式**，2022 年 2 月正式启动。核心特征是：**一个项目经理从头到尾负责全部升级工程**，业主只需对接一家公司。

**OSS 负责的工作**：
1. 初始 Home Energy Assessment（HEA）
2. 制定升级方案 + 技术设计
3. 申请全部 SEAI 补贴
4. 统筹所有施工单位
5. 施工质量监督
6. 完工后 BER 复评

### 2.2 OSS 模式下可申请的补贴金额（2026 年）

| 升级项目 | 单独申请补贴 | OSS 模式补贴 | 说明 |
|----------|------------|-------------|------|
| 外墙保温 (External Wall) | €3,500-€8,000 | 更高（含入套餐） | 独立屋 vs 半独/联排 |
| 内墙保温 (Internal Dry Lining) | €2,000-€4,500 | 更高（含入套餐） | |
| 空心墙保温 (Cavity Wall) | €1,500-€2,500 | 更高（含入套餐） | |
| 热泵 (Heat Pump) | €6,500 | **€12,500**（2026 新标准） | OSS 翻倍 |
| 太阳能 PV | €1,800-€2,100 | 更高 | |
| 太阳能热水 | €1,200 | 同单独 | |
| 窗户/门 | 无单独补贴 | **有补贴**（2026 新增） | 仅 OSS 模式下 |
| OSS 项目管理费 | — | €2,000 | 专为 OSS 模式 |
| **单户最高补贴总额** | ≈€27,000 | **≈€36,900+** | 含 OSS 专属 bonus |

> 数据来源：SEAI 2026 年 2 月 3 日生效的新政；热泵补贴上调至 €12,500，门窗首次纳入补贴范围。

### 2.3 OSS 升级的五大核心项目

#### 2.3.1 墙体保温

| 保温类型 | 适用房屋 | 单位成本 | 施工周期 | 节省预估 |
|----------|----------|---------|---------|---------|
| 空心墙填充 (CWI) | 2000 年后建造 | €8-€12/m² | 1-2 天 | 20-30% 供暖 |
| 外墙保温 (EWI) | 实心墙/旧房 | €120-€180/m² | 2-4 周 | 30-40% 供暖 |
| 内墙干衬 (IDL) | 保护建筑 / 受限 | €70-€110/m² | 1-3 周 | 25-35% 供暖 |

**华人 tips**：独立屋外墙面积约 120-160m²，EWI 预算 €14,000-€28,000，补贴后净支出约 €10,000-€20,000。

#### 2.3.2 热泵系统（Air-to-Water Heat Pump）

- **市场价**：€14,000-€20,000（含安装 + 散热器替换）
- **OSS 补贴**：€12,500（2026 新标准）
- **净支出**：€1,500-€7,500
- **年节省**：相对于燃油锅炉约 €800-€1,200/年
- **前提条件**：房屋 Heat Loss Indicator（HLI）≤ 2 W/K/m²，通常需先做保温

#### 2.3.3 太阳能光伏（Solar PV）

| 系统容量 | 面板数 | 总成本 | 补贴 | 净支出 | 年发电量 | 年节省 |
|----------|--------|--------|------|--------|----------|--------|
| 3kWp | 8-10 块 | €5,000-€6,500 | €1,800 | €3,200-€4,700 | 2,700 kWh | €370-€400 |
| 5kWp | 12-14 块 | €7,500-€9,500 | €2,100 | €5,400-€7,400 | 4,500 kWh | €550-€600 |
| 7kWp | 16-18 块 | €9,500-€12,000 | €2,100 | €7,400-€9,900 | 6,300 kWh | €700-€800 |

加装电池（5-10kWh）：额外 €3,000-€6,000，无单独补贴。

#### 2.3.4 窗户和门（2026 年新纳入补贴）

- **类型**：双层 Low-E 或三层玻璃，uPVC / 铝包木 / 复合材质
- **窗户单价**：€400-€800/扇（含安装）
- **外门单价**：€800-€2,000/扇
- **全屋预算**：15 扇窗户 + 2 扇门 ≈ €8,000-€16,000
- **补贴**：仅限 OSS 模式，金额取决于项目整体

#### 2.3.5 阁楼保温

- **标准厚度**：300mm（Building Regs 要求）
- **成本**：€1,200-€2,000（排屋）/ €2,000-€3,500（独立屋）
- **补贴**：€800-€1,500（OSS 模式）
- **节省**：供暖费用降低 15-20%

### 2.4 SEAI 补贴核心数据（2026 年上半年）

来自 SEAI 统计数据：
- 2026 年 H1 共完成 **21,764 例** 能源升级
- OSS 模式下占比约 40%
- 平均每户总补贴：€21,000（含 bonus）
- 平均完工 BER：从 D1 提升至 B2

### 2.5 OSS 升级时间线

```
第 1 周：选择 OSS，签合同，付 HEA 费用（通常 €350-€600）
第 2-3 周：OSS 上门做 Home Energy Assessment
第 4-5 周：收到设计方案 → 确认 → 签施工合同
第 6 周：OSS 提交 SEAI 补贴申请（通常 2-3 周批复）
第 7-8 周：施工队伍进场，保温工程
第 9-10 周：窗户替换、新风安装
第 11-12 周：热泵 + 太阳能安装
第 13-14 周：收尾、完工 BER 评估
总周期：3-6 个月（取决于 OSS 排期和房屋复杂度）
```

**华人 tips**：
- 施工前 2-3 个月开始联系 OSS，不要等拿到钥匙才找
- 旺季（夏季）排期 >> 淡季（冬季），但保温施工冬季影响不大
- 如果同时做装修（厨房/卫生间/地板），要协调 OSS 与装修队的进场顺序

## 第三章：中国无证装修队如何办 BER

### 3.1 核心矛盾

中国装修队在爱尔兰普遍不持有以下资质：
- SEAI 注册承包商证书
- NSAI 施工认证
- Safe Electric / RGII 等专业资质
- 英/爱本地保险（Public Liability Insurance）

**直接后果**：SEAI 不会直接向无证承包商支付补贴。

### 3.2 解决方案：四条路径

#### 路径一：混搭模式（推荐）

| 角色 | 任务 | 执行方 |
|------|------|--------|
| OSS 公司 | HEA、方案设计、补贴申请、技术监督、完工 BER | 注册 OSS |
| 中国装修队 | 室内装修、地板、厨房/卫浴、油漆、木工 | 无证 |
| OSS 合作施工队 | 保温、热泵、太阳能、窗户、电气 | SEAI 认证 |

**操作要点**：
1. 签约 OSS 时明确告知"室内装修另有安排"（多数 OSS 接受）
2. OSS 的施工队负责所有补贴覆盖的能效项目
3. 中国装修队在 OSS 项目之前或之后进场（需协调顺序）
4. 总费用：OSS 报价 + 装修队报价，补贴直接抵扣 OSS 部分

**优点**：补贴全拿 + 装修费用可控（中国队通常便宜 30-50%）
**缺点**：需协调两队衔接

#### 路径二：中国包工头挂靠本地 OSS

有一定可能性的方案，适合在爱尔兰已有多年积累的中国包工头：

1. 注册一家爱尔兰公司
2. 聘请 SEAI 注册 Technical Advisor（TA）作为顾问/雇员
3. 将保温、热泵、太阳能等 SEAI 必须认证的环节分包给本地注册承包商
4. 自己承担项目管理、装修和施工协调

**挑战**：
- 公司注册 + SEAI OSS 注册流程约 6-10 个月
- 需配备专业保险（€2,000-€5,000/年）
- SEAI 对 OSS 的资质审查严格，首次申请人需提交 2 年以上商业计划

#### 路径三：纯 DIY + 分项申请补贴（不推荐）

不通过 OSS，自己找 SEAI 注册承包商逐项施工，逐项申请补贴：

- 补贴金额更低（无 OSS bonus）
- 每个项目需单独走 TA 评估 → 申请 → 施工 → 验收
- 热泵必须先做 TA 评估（€500-€800）
- 窗户/门无单独补贴

**适用**：只做 1-2 项升级、不追求全屋翻新的人。

#### 路径四：整包给 OSS（最省心）

全部交给 OSS 做，装修部分如果 OSS 能一起包则包，不能则 OSS 做完能效升级后再找人装修。

- 补贴最大化
- 协调最少
- 但 OSS 价格通常高于拆分模式

### 3.3 华人装修队可做 vs 不可做

| 可以自己做 | 必须找认证人员 |
|-----------|--------------|
| 室内隔墙 | 外墙/内墙保温（需 SEAI 认证） |
| 厨房/卫浴安装 | 热泵系统安装 + 调试 |
| 地板铺设 | 太阳能 PV 并网 + 电气连接 |
| 油漆/装饰 | 新窗户/门安装（如需补贴） |
| 内部门 | 新风系统 MVHR 设计安装 |
| 非结构性木工 | 任何涉及电气、燃气的项目 |
| 石膏板/Dry Lining | BER 评估和盖章 |

### 3.4 无证队做保温的"灰色地带"

技术上中国工人完全可以贴外墙保温板（EWI），但：
- SEAI 补贴要求施工方有 NSAI 认证，否则无法拿补贴
- 如果不要补贴，可以自己做，但完工 BER 不会改善
- **实际常见做法**：外包保温给本地认证公司（走补贴），室内装修用中国队

### 3.5 重点提醒

> ⚠️ **电气和燃气工程**无论是否要补贴都必须由 Safe Electric / RGII 注册人员完成。这是法律要求（Building Regulations），不是补贴要求。中国装修队没有任何例外。

## 第四章：询价全流程与省钱策略

### 4.1 询价前的准备工作

**Before 联系任何一个 OSS，你需要准备好**：

1. **Eircode**（必须，定位房屋）
2. **现有 BER 证书**（如果有）
3. **房屋基本信息**：
   - 面积（m²）
   - 建造年份
   - 房屋类型（独立/半独/联排/公寓）
   - 墙体类型（空心砖/实心墙/预制板）
   - 现有供暖方式（燃气/油/电/其他）
4. **你的目标**：全屋 OSS 升级 / 分项升级 / 仅咨询
5. **预算范围**（不用精确，有个概念即可）

### 4.2 询价邮件模板（中英文）

**英文模板**：

```
Subject: Home Energy Upgrade Enquiry – [Eircode]

Dear [OSS Company Name] Team,

I recently purchased a property at [Address], Eircode [Eircode], and am
interested in your One Stop Shop service.

Property details:
- Type: [Detached/Semi-detached/Terraced/Apartment]
- Floor area: ~[XXX] m²
- Year built: [YYYY]
- Current heating: [Gas/Oil/Electric]
- Current BER (if known): [e.g., D2]

I am looking for a whole-house energy upgrade to achieve at least B2
BER. Could you please provide:
1. An outline of your process and timeline
2. A rough indicative cost estimate (pre-grant and post-grant)
3. Whether you can accommodate separate interior renovation works
   arranged by myself

I can be reached at [phone number] or this email.

Many thanks,
[Your Name]
```

**中文说明**：
- 用英文发邮件即可，大部分 OSS 没有中文服务
- 如果英语不好，建议用 DeepL / ChatGPT 翻译后发送
- 电话沟通时可以要求对方放慢语速

### 4.3 如何对比 OSS 报价

**至少要拿到 3 家报价**，对比维度：

| 项目 | OSS A | OSS B | OSS C |
|------|-------|-------|-------|
| HEA 费用 | | | |
| 总报价（税前） | | | |
| 预扣补贴金额 | | | |
| 实付金额 | | | |
| 是否包窗户 | | | |
| 是否包室内恢复 | | | |
| 排期 | | | |
| 完成时间 | | | |
| 质保期 | | | |
| 是否支持外部装修队 | | | |

**华人特别提示**：
1. 问清楚报价是否包含"Making good"（恢复施工后的墙面/地面/油漆）
2. 多数 OSS 只做到"skim coat"级别，不包含全屋油漆、地板、踢脚线
3. 热泵替换锅炉后旧油罐/锅炉拆除是否额外收费
4. BER 达到 B2 是合同约束的交付标准（完工不达标应免费整改）

### 4.4 省钱策略

#### 策略一：HEA 费用可讲价
- HEA 市价 €350-€600，部分 OSS 在签施工合同后会返还
- 问清楚：是否可以在签合同后退 HEA 费

#### 策略二：捆绑 OSS + 单独补贴
- OSS 升级整体报价，但实际施工可能分包给多个认证施工队
- 如果某项 OSS 报价明显偏高（比如太阳能），可以选择自己找认证安装商，走单独补贴

#### 策略三：HEULS 低息贷款
- 2024 年推出的 Home Energy Upgrade Loan Scheme
- 无抵押贷款，额度 €5,000-€75,000
- 利率显著低于市场商业贷款
- 通过参与银行（BOI, PTSB, AIB 等）申请
- 还款期最长 10 年

#### 策略四：装修队与 OSS 协调
- OSS 做能效升级 → 装修队入场 → OSS 回来做 final snagging
- 这样装修队不干扰 SEAI 验收，还能用自己的工人做后续

#### 策略五：窗户可拆分
- OSS 报价中的窗户通常用中档品牌（如 Munster Joinery / Senator）
- 如果你想要更高端的（如 Internorm / Rationel），可以自购窗户、请 OSS 仅做安装
- 需事先协商

### 4.5 常见收费陷阱

| 陷阱 | 表现 | 对策 |
|------|------|------|
| 隐藏恢复费 | 报价不含"making good"，施工后一片狼藉 | 合同明确包含范围 |
| 补贴预扣猫腻 | 声称可以拿到补贴但实际申请不下来 | 合同写明：补贴不到位由 OSS 承担差额 |
| BER 不达标补差价 | 完工后 BER 差一点到 B2，OSS 要求加钱 | 合同中 B2 作为交付条件，不达标免费整改 |
| 临时加项 | 施工中发现"需要额外工程" | 签合同前让 OSS 做充分 HEA，减少不确定性 |
| 质量隐患 | 验收时看起来不错，半年后出问题 | 要求 OSS 写明质保期和响应时间 |
