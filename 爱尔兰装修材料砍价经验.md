# 一、浴室材料 Brooks / Sonas / Chadwicks 三家渠道对比

## 1. 核心对比表

| 对比维度 | Sonas Bathrooms | Chadwicks | Brooks Timber & Building Supplies |
|----|----|----|----|
| 核心定位 | 爱尔兰专业卫浴品牌，独立大型卫浴展厅，全套浴室整装方案 | 综合工程建材批发商，卫浴+水管+保温一站式，面向包工头与散户 | 木材、保温、门窗为主的综合建材商，卫浴仅配套基础单品 |
| 产品层级 | 中端-高端成套卫浴，20+套系浴室柜、淋浴房、浴缸、恒温花洒齐全 | 中端工程款洁具，款式实用简约，网红款式偏少 | 平价出租房基础马桶、面盆，无定制浴室柜、独立淋浴房产品线 |
| 票据合规性 | 全额VAT发票、产品质保单据齐全，完美适配SEAI补贴申报 | 全品类增值税票、送货单、质保凭证，本地施工队主流拿货渠道 | 可开正规发票，但卫浴配套品类极少 |
| 配套服务 | 免费上门勘测、浴室3D布局设计、专属卫浴顾问、整套搭配方案 | 全品类建材技术顾问，水管/采暖/防水一体化咨询，全城配送能力强 | 无卫浴设计师，仅现货售卖，无配套设计服务 |
| 适配人群 | 自住全屋翻新、多卫生间改造、追求成套颜值、需要规范票据 | 大面积全屋翻新、水电暖通同步采购、包工头批量拿货、一站式囤货 | 低成本单间出租简易卫浴更换，仅替换基础洁具 |
| 短板 | 小件五金配件单价偏高，瓷砖、木材、保温需单独去别家采购 | 浴室设计感弱，极简、复古网红款式稀缺 | 卫浴款式老旧，选择面窄，无法做整套浴室搭配 |

## 2. Sonas Bathrooms 完整采购流程（都柏林 D15 主展厅）

1. 预约到店咨询：官网/电话预约全国主展厅（103 NW Business Park, Ballycoolin D15），免费一对一卫浴方案咨询，现场实景样板间选款
2. 上门免费勘测：提供房屋地址、卫生间长宽、下水、门窗位置，专员上门测量，规避80-90年代老房异形墙体尺寸问题
3. 出具明细报价：设计师输出平面布局图、洁具清单、含税总价，区分散户零售价、包工贸易折扣价
4. 议价下单：全屋多卫生间批量采购可谈贸易折扣，下单支付30%定金，留存完整报价单、产品型号、官方质保年限
5. 备货周期：标准现货洁具3–7天；定制浴室柜4–6周
6. 送货验收：支持都柏林全城送货，收货当场核对型号、检查陶瓷磕碰，破损现场拍照直接换货
7. 票据归档：索取全额VAT发票、产品质保卡、送货签收单，统一存档用于SEAI节能补贴、卖房退税
8. 售后质保：全系洁具5–10年官方质保，漏水、五金故障凭全套单据免费上门检修

# 二、瓷砖板块：价格区间 + 7家都柏林瓷砖门店对比

## 1. 2026都柏林瓷砖含税单价区间（€/㎡）

1. 出租房基础哑光瓷片：€9.99–19.99
2. 自住普通通体大理石600×600/600×1200：€20–40
3. 柔光木纹砖、水磨石、中大型规格通体砖：€35–55
4. 高端岩板、人字拼艺术砖、进口特色砖：€55–90
5. 户外庭院防滑地砖：€25–40
6. 马赛克、六角复古小砖、维多利亚花砖：€45–80

## 2. 都柏林7家瓷砖门店完整对比

1. **Tile Merchant（Coolock/Ballymount/Deansgrange/Ashbourne 4分店）**
   - 优势：本土大型连锁，现货库存量大，基础砖底价€9.99起；满额免费送货，正规VAT发票齐全，自住+出租全覆盖
   - 短板：高端进口岩板款式有限，无配套室内设计服务
   - 适配：全屋厨卫、地面通铺大面积采购，性价比首选
2. **City Tiles & Bathrooms（Ballymount D24）**
   - 优势：上万平超大展厅，瓷砖+卫浴一体化展示，50+实景浴室样板间，一站式配齐厨卫全套材料
   - 短板：散户单独采购议价空间极小，整体单价偏高
   - 适配：自住高端全屋翻新，一次性搞定瓷砖+洁具
3. **Waterhouse Tiles（Long Mile Road D12）**
   - 优势：平价复古花砖、马赛克、维多利亚小砖库存充足，特色造型砖单价偏低
   - 短板：大尺寸岩板现货少，免费送货门槛高
   - 适配：玄关、特色卫生间局部造型改造
4. **Doherty Flooring（Blanchardstown D11）**
   - 优势：瓷砖+木地板同店采购，配套楼梯踏步收口条齐全，防潮方案适配老房底层潮湿区域
   - 短板：瓷砖品类少于专业瓷砖店，以地面砖为主，墙面瓷片款式少
   - 适配：地板、瓷砖同步采购的全屋翻新项目
5. **Chadwicks 全部分店**
   - 优势：工程级耐磨基础瓷砖库存充足，包工头贸易账户折扣力度大，瓷砖、水管、保温可统一结算开票
   - 短板：款式偏工装极简，柔光、网红艺术砖极少
   - 适配：投资出租房大批量基础瓷砖囤货
6. **B&Q（Blanchardstown/Tallaght/Swords）**
   - 优势：门店分布广，随到随选，少量补货便捷，填缝剂、找平辅料一站式配齐
   - 短板：全屋大面积采购无价格优势，瓷砖耐磨度一般
   - 适配：局部修补、单间简易出租改造
7. **华人建材仓库（Ballycoolin/Tallaght工业区）**
   - 优势：中文沟通无障碍，可打包瓷砖+搬运同步安排，总价灵活
   - 避坑：必须主动索要完整VAT发票，无票店铺无法申报SEAI补贴
   - 适配：包工头统一打包全屋建材批量采购

> 补充说明：实际案例中瓷砖由包工头统一走贸易账户批量采购，拿到散户无法谈下的低价；散户单独采购优先 Tile Merchant、Waterhouse 议价。

# 三、电器与工具：Lidl/Aldi vs Woodie's/B&Q 价格&品类对比

## 基础价格定位
- Lidl/Aldi：平价自有工具线，整体价格比 Woodie's/B&Q 低40%–60%，性价比拉满
- Woodie's / B&Q：专业一线品牌工具（博世、DeWalt、Stanley）齐全，官方完整质保，单品售价更高

## 分渠道优势品类

### Lidl（Parkside 工具线，装修电动工具首选）
1. 优势品类：锂电冲击钻、角磨机、砂光机、吸尘器、木工切割工具、卷尺、夹具、花园电动工具
2. 核心优势：同规格电动工具价格仅为 Woodie's 一半，自住翻新完全够用
3. 局限：无精密电工、水工专业工具，活动不定期，断货补货周期长

### Aldi（Ferrex 工具线，油漆辅料刚需）
1. 优势品类：全套油漆辅料（滚筒、毛刷、保护膜、美纹纸、修补腻子）、各类密封胶 sealant、玻璃胶、厨卫防水胶、小型手动打磨工具、一次性施工防护服
2. 核心优势：密封胶、油漆耗材单价远低于连锁DIY门店，厨卫收口耗材优先在这里囤货

### Woodie's / B&Q（B&Q 全称 Block & Quayle）
1. 优势品类：全系列专业工业工具、品牌油漆、全屋门窗五金、大型花园设备、暖通配套小配件；可调色墙面漆、防霉专用漆款式齐全
2. 核心优势：全时段稳定供货，质保体系完善，技术店员可提供基础施工建议，门店多补货方便
3. 短板：电动工具、密封胶、油漆辅料溢价极高，长期全屋翻新采购成本大幅高于商超折扣款

## 采购建议
- 电动施工工具、手动五金 → Lidl
- 玻璃胶、密封胶、油漆全套耗材 → Aldi
- 品牌专业工具、调色漆、应急少量补货 → Woodie's / B&Q

# 四、水管材料：四大主流采购渠道对比

1. **Chadwicks（工程批发商，首选全屋翻新）**
   - 主营：PEX水管、冷热管件、暖气片、地暖管材、下水排污管、水泵、阀门全套水暖配件
   - 优势：品类全覆盖，本地持证水工主流拿货渠道，贸易账户折扣大，可开完整VAT发票，同步搭配保温、卫浴一站式采购，全城配送
   - 适配：全屋水电改造、热泵/MVHR暖通同步施工、大面积翻新
2. **Plumb Merchants Ltd（D1市中心专业水暖专营）**
   - 主营：家用供水、供暖、卫浴下水专用管材配件，细分型号齐全，暖通维修小件储备充足
   - 优势：水暖专业技术人员，适配老房管道改造，市中心补货便捷
   - 适配：局部水管更换、维修补件、小户型卫生间翻新
3. **Screwfix（连锁水暖工具店，DIY自装散户）**
   - 主营：标准化家用水管、快速接头、简易下水配件、小型维修工具
   - 优势：线上线下同步，当日取货，小件单价透明
   - 短板：大批量全屋改造无低价，暖通大型管材品类少
   - 适配：自装散户、局部维修、少量补件
4. **华人建材仓库（工业区）**
   - 优势：沟通无障碍，管材、辅料打包报价灵活，可同步安排搬运
   - 避坑：优先索要VAT发票；部分仓库管材无官方质保，暖通改造不建议大批量采购
   - 适配：出租房简易水电改造，预算极低短期过渡项目

## 渠道选择逻辑
- 全屋翻新、做热泵/地暖、需要合规票据申报补贴 → Chadwicks
- 市中心局部维修、小件补料 → Plumb Merchants / Screwfix
- 低成本出租简易改造，不考虑长期质保与补贴 → 华人仓库

# 五、其他全品类建材补充（地板/油漆/门窗/保温/花园）

## 1. 地板采购渠道

1. 自住全屋、可开票申报补贴：Doherty Flooring(D11)、Dublin Floors and Doors(D24)、Unique Flooring
2. 批发自装省钱：H&F Trade Floors（B2B批发，仅供货不含安装）
3. 低成本出租DIY：Mulveys、B&Q、IKEA Ballymun
4. 底层潮湿厨卫专用：Irish Industrial Floor Company(D3 East Wall)防水LVT/PVC地坪

> 采购要点：老房底层/朝北房间优先SPC防水地板；全屋采购最优方案：门店买主材，单独聘请持证木工安装，规避包工加价。

## 2. 油漆采购渠道

1. Aldi：密封胶、滚筒、毛刷、腻子等耗材最便宜，常备基础哑光内墙漆
2. Woodie's/B&Q：多品牌成品漆、可调色漆、防霉专用漆、户外木漆款式最全，适合追求色彩定制、防潮防霉需求
3. Chadwicks/McMahons：工程大桶内墙漆，全屋大面积翻新批量拿货单价最低，可开票

> 避坑：潮湿老房厨卫、底层必须选用Zinsser防霉底漆，优先在DIY连锁门店采购。

## 3. 门窗采购渠道

1. Sonas、Chadwicks：配套入户门、室内门、淋浴房玻璃门，同步卫浴改造一站式采购
2. 专业门窗厂：本地持证门窗定制商家，双层节能断桥铝门窗，可拿SEAI门窗补贴，正规发票齐全
3. B&Q/IKEA：标准化室内门，低价简易款，仅适合出租房低成本替换

> 要点：想要提升BER评级、申报补贴，必须选可开具VAT发票的持证门窗厂家，单层玻璃门窗不建议自住长期使用。

## 4. 保温材料采购渠道（BER提分核心）

1. Chadwicks / Brooks Timber：全屋内墙、阁楼外墙保温棉、保温板，工程批量价最低，配套防潮隔气膜全套材料，施工队主流渠道，票据合规适配SEAI补贴
2. McMahons Builders Providers：各类保温板材现货充足，都柏林多分店，配合SEAI一站式申报方案
3. 专业保温公司：BER Assessor 推荐的本地持证保温安装商，全包方案含材料+施工+SEAI补贴申报

> ⚠️ 保温不可DIY，必须由SEAI注册施工方安装才能申请补贴。

## 5. 花园/户外材料采购渠道

1. Woodie's / B&Q：户外地板、围栏、工具、植物、花园家具，品类最全
2. Chadwicks / Brooks：铺设车道碎石、水泥、铺路砖、排水管道工程材料
3. Lidl / Aldi：季节性花园工具、花盆、户外小家具，价格极低

# 六、通用砍价心法总结

1. **先拿报价再砍价**：先去 Sonas/Brooks/Chadwicks 各拿一份明细报价，用A家的报价压B家
2. **批量采购谈贸易折扣**：全屋翻新一次性采购量大的，申请开通贸易账户（Trade Account），折扣力度显著
3. **Cash deal 问折扣**：部分中小门店现金支付可再谈 5%-10% off
4. **避开旺季**：春季（3-5月）和夏季是装修旺季，价格坚挺；秋冬采购建材议价空间更大
5. **包工头代采**：如果包工头有 Chadwicks/Brooks 贸易账户，让他代买主材，能拿到散户根本拿不到的价格
6. **VAT发票是命根子**：所有大额采购必须索要VAT发票，不要为了省几个点放弃发票——SEAI补贴、卖房退税全靠它
7. **购买顺序建议**：先定卫浴（Sonas）→ 再定瓷砖（Tile Merchant）→ 水管保温（Chadwicks）→ 工具耗材（Lidl/Aldi）→ 地板门窗 → 油漆最后
